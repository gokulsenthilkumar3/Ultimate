# Security Policy

## Supported Versions

Only the current major version is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within GrowthTrack Ultimate, please DO NOT open a public issue.
Instead, please send an e-mail to the maintainer. All security vulnerabilities will be promptly addressed.

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress.
If a fix is available, we will notify you before the patch is released.
