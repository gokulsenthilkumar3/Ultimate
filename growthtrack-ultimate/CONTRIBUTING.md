# Contributing to GrowthTrack Ultimate

Thank you for your interest in contributing to GrowthTrack Ultimate! We welcome contributions from everyone.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/Ultimate.git`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`

## Branching Strategy

- `main`: Stable production code
- `develop`: Integration branch for new features
- Feature branches should be named `feature/<feature-name>`
- Bugfix branches should be named `bugfix/<bug-name>`

## Submitting Pull Requests

1. Ensure your code passes all linting (`npm run lint`) and tests (`npm run test`).
2. Include a descriptive summary of your changes.
3. Link the PR to the relevant GitHub Issue.

## Code Style

- We use ESLint and React Hooks plugin.
- Please follow the existing code structure.
- Document any new features or configurations in the README.
